# CarFlow 名称辞書（共通認識ドキュメント）

このドキュメントは、UI 上に出る用語と内部で使う識別子の **正式名称** を 1 か所にまとめたものです。
ブレた呼び方を避けるため、変更や議論はまずこのファイルを更新してから実装に反映します。

最終更新: v0.7.8

---

## 凡例

- **正式名称**：UI に表示する正規ラベル
- **別名 / よくあるブレ**：過去に使われた・口頭で出る別表記（このドキュメントで統一）
- **内部 ID / 変数名**：HTML / JS / CSS で使う識別子（変えると検索で追いきれなくなる箇所）
- **要相談**：ゆうたさんと統一が必要な箇所には ⚠️ を付けています

## 10. 用語の定義（混乱しがちな単語）

| 用語 | 定義 |
|---|---|
| 在庫日数 | 仕入れ日（`purchaseDate`）から今日までの日数 |
| 売約日数 | 売約日（`contractDate`）から今日までの日数 |
| 納車まで N 日 | 納車予定日（`deliveryDate`）までの残日数。マイナスは「納車超過」 |
| 警告色 3 段 | 緑 (dg) → 橙 (dw) → 赤 (dr)。閾値は設定で調整 |
| リードタイム | 売約日から納車予定日までの**初期日数**（設定で変更可） |
| 月次集計締め | その月の販売実績を確定する操作 |
| 販売実績 | 月次で締めた過去の販売履歴 |

## 11. 命名ルール（今後の追加用）

- **HTML 要素 ID**：ケバブケース（例：`si-dashboard`, `panel-settings`）
- **CSS クラス**：ケバブケース＋プレフィックス（`cc-` = car-card 系、`sb-` = sidebar 系、`wf-` = workflow 系）
- **JS 関数**：キャメルケース（例：`renderKanban`, `closeSellConfirm`）
- **JS 定数**：UPPER_SNAKE_CASE（例：`REGEN_TASKS`, `COLS`）
- **メモ系プロパティ**：`car.memo`（コアメモ） / `car.workMemo`（作業メモ）
